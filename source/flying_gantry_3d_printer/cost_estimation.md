##  Build Notes

- The frame uses **aluminum extrusion** for rigidity  
- Motion system includes **linear rail + V-wheel support system**  
- Heated bed with **magnetic flexible plate** for easy print removal  
- Designed as a **prototype-level 3D printer system**

##  Remarks

This project demonstrates the integration of **mechanical design, embedded systems (RP2040), and motion control** in a functional 3D printing setup.
