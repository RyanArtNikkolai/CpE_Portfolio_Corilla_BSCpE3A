## 📌 Project Description

**SafeVoice** is a mobile application designed to help users report abuse incidents quickly and easily. The main goal of the system is to provide a simple and accessible way for users to send reports, especially during emergency or unsafe situations.

This project focuses on improving safety and awareness by allowing users to share incident details that can be used for response and documentation.

---

## 🎯 Objectives

- To provide a fast reporting system for abuse incidents  
- To improve user safety through easy access to reporting tools  
- To practice mobile application development skills  
- To demonstrate real-world problem solving using technology  

---

## 🧰 Key Features

- User-friendly reporting interface  
- Incident report submission  
- Location sharing (if implemented)  
- Simple and fast navigation  
- Basic emergency reporting system
