#  Audio Amplifier Project

##  Project Description

This project demonstrates a basic audio amplifier circuit using a TIP41 NPN power transistor to amplify low-level audio signals from devices such as phones or computers. The project is designed for educational purposes, DIY electronics applications, and classroom demonstrations.

The audio signal first passes through a potentiometer that acts as a volume control. The TIP41 transistor is configured in a common-emitter amplifier mode where a small signal at the base controls a larger current flowing from the collector to the emitter. This process amplifies the weak audio signal into a stronger output signal.

The circuit is powered using a 9V–12V DC power supply which provides the necessary power for audio amplification.
