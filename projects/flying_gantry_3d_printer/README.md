#  Flying Gantry 3D Printer 

---

##  Project Description

This project is a **Flying Gantry 3D Printer System** designed as part of our Computer Engineering activities. It focuses on improving 3D printer performance by using a flying gantry mechanism that makes movement smoother and more stable.

The system is controlled using the **RP2040 microcontroller (Raspberry Pi Pico)**, which handles the stepper motor movements for accurate positioning during operation.

---

##  Project Objective

- To design a stable and efficient 3D printer mechanism  
- To reduce vibration using a flying gantry structure  
- To use RP2040 for controlling motor movements  
- To apply embedded systems and mechanical design in a real project  

---

##  Components Used

- RP2040 Microcontroller (Raspberry Pi Pico)  
- Stepper Motors  
- Stepper Motor Drivers (A4988 / DRV8825)  
- 3D Printed Parts (Flying Gantry Structure)  
- Linear rods / rails  
- Belts and pulleys  
- Power supply  
- End stop
---

##  How It Works

The flying gantry design reduces the weight of moving parts, allowing smoother and more stable motion during printing.

The RP2040 sends signals to the motor drivers, which control the stepper motors. This enables precise movement along different axes for better accuracy and print quality.

---

##  Features

- Flying gantry mechanical design  
- Smooth and stable motion system  
- RP2040-based motor control  
- Lightweight 3D printed structure  
- Modular and upgradeable design
